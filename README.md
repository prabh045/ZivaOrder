# ZivaOrder
An orderTaking application

Features
1. Made using MVVM and clean architecture
2. Unit Tests written
3. Core Data implemented for local storage
4. UI made programmatically using autolayout

Third Part used

-Kingfisher Installed using SPM

Installation
1. Clone the repo
2. open .xcodeProject
3. Make sure that Kingfisher is there, otherwise please install using any preferred method
